#### 在每个树行中找最大值/Find Largest Value in Each Tree Row
**难度：** 中等/Medium

**Question：** 

<p>You need to find the largest value in each row of a binary tree.</p>

<p><b>Example:</b><br />
<pre>
<b>Input:</b> 

          1
         / \
        3   2
       / \   \  
      5   3   9 

<b>Output:</b> [1, 3, 9]
</pre>
</p>


------
